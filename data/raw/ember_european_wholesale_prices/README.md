# European Wholesale Electricity Prices – Monthly

## Source
Ember – European Wholesale Electricity Price Data  
https://ember-energy.org/data/european-wholesale-electricity-price-data/

## Description
This dataset contains average monthly wholesale day-ahead electricity prices for European countries. 

Note that these are the prices generators receive for selling electricity on the spot market. They are not the same as the prices paid by electricity consumers, which can also include taxes, levies, network charges, subsidies, and supplier profits. They also do not account for hedging. For an explanation of how spot prices are formed, see [here](https://neon.energy/marginal-pricing#:~:text=Marginal%20pricing%20refers%20to%20electricity,as%20the%20day%2Dahead%20market.).

## Structure
Main columns:
- Country
- ISO3 Code
- Date
- Price (EUR/MWhe)

## Notes
- Prices represent spot market prices received by generators.
- Prices do not include taxes, levies, network charges or supplier margins.
- Data is aggregated from hourly values and weighted by load.
